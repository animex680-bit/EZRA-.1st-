# Fonts Library
## All fonts used in the 0x100x style pipeline

---

## Tier 1 — Primary (Must Have)

### Druk Wide Bold
- **Style role:** Impact headlines, main lyric text, statement words
- **Weight used:** Bold, Super (heaviest)
- **Case:** ALL CAPS exclusively
- **License:** Commercial — purchase at commercialtype.com
- **File name when acquired:** `DrukWide-Bold.otf`, `DrukWide-Super.otf`
- **Free substitute:** Bebas Neue (see below) — use when Druk is not licensed

### Bebas Neue
- **Style role:** Impact headlines (free substitute for Druk Wide)
- **Weight used:** Regular (it's only one weight — ultra-condensed bold)
- **Case:** ALL CAPS
- **License:** Free — Google Fonts commercial use allowed
- **Download:** fonts.google.com → search "Bebas Neue"
- **File name:** `BbebasNeue-Regular.ttf`
- **Note:** Slightly narrower than Druk — increase tracking slightly to compensate

### Inter
- **Style role:** Supporting text, subheadings, captions
- **Weights used:** Medium (500), SemiBold (600), Bold (700)
- **Case:** Mixed case or ALL CAPS
- **License:** Free — Google Fonts (SIL Open Font License)
- **Download:** fonts.google.com → search "Inter"
- **File names:** `Inter-Medium.ttf`, `Inter-SemiBold.ttf`, `Inter-Bold.ttf`

## Tier 2 — Secondary (Use Often)

### Anton
- **Style role:** Alternative to Bebas Neue for condensed bold headlines
- **Weight used:** Regular (single weight — very bold)
- **Case:** ALL CAPS
- **License:** Free — Google Fonts
- **Download:** fonts.google.com → search "Anton"
- **File name:** `Anton-Regular.ttf`

### Space Mono
- **Style role:** Tech/crypto counters, timestamps, UI readouts, monospaced text
- **Weights used:** Regular (400), Bold (700)
- **Case:** UPPERCASE for numbers and codes; mixed for labels
- **License:** Free — Google Fonts
- **Download:** fonts.google.com → search "Space Mono"
- **File names:** `SpaceMono-Regular.ttf`, `SpaceMono-Bold.ttf`

### Neue Haas Grotesk Display Pro
- **Style role:** Premium secondary text when Druk is the headline
- **Weights used:** Light, Regular, Medium, Bold
- **License:** Commercial — Linotype / Monotype
- **Note:** If not licensed, use Inter Bold as substitute

## Fonts NOT to Use in This Style

These fonts break the aesthetic immediately:

| Font | Why not |
|---|---|
| Comic Sans | Obviously no |
| Lobster | Too casual, too rounded |
| Pacifico | Too playful |
| Times New Roman / Georgia | Serif doesn't fit this aesthetic |
| Arial / Calibri | Office aesthetics — break the premium feel |
| Futura (light weights) | Too thin — disappears on dark backgrounds |
| Script fonts (Parisienne, etc.) | Completely off-brand |

## Font Installation Notes

**On Mac:**
1. Download .ttf or .otf file
2. Double-click → Font Book → Install Font

**On Windows:**
1. Download .ttf or .otf file
2. Right-click → Install for all users

**In After Effects:**
Fonts installed to the OS are automatically available in AE.
If After Effects is open when you install, restart AE to see new fonts.

**In DaVinci Resolve:**
Same — OS-level installation, then restart Resolve.
