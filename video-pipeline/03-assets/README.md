└── sfx/

## Physical Asset Storage Rule

Every physical asset file (fonts, LUT .cube files, overlay .mov files, SFX .wav files) should be stored in:

1. **This repo** — under `03-assets/[category]/files/` (add files to git)
2. **Per-project** — a copy placed in the project's `05_ASSETS_USED/` folder for offline archiving

When you acquire a new asset that doesn't exist in the library, add it to:
- The physical file location above
- The relevant list document below

This way the library grows with every project.

## Quick Download Priority

If starting from zero, acquire these first (all free):

| Asset | Where to get | Priority |
|---|---|---|
| Bebas Neue font | Google Fonts | Immediate |
| Inter font | Google Fonts | Immediate |
| Space Mono font | Google Fonts | Immediate |
| Film grain overlay footage | actionvfx.com (free tier) or Pexels | High |
| Phantom LUT | AEJuice community / search "Phantom LUT free" | High |
| Transition whoosh SFX | Freesound.org | High |
| Impact boom SFX | Freesound.org | High |
| Twixtor Pro plugin | RE:Vision Effects ($) | High (essential for speed ramps) |

## Licensing Notes

- Always verify licensing before commercial use
- Google Fonts = free for commercial
- AEJuice free tier = personal use; upgrade for commercial
- Twixtor Pro = paid commercial license required
- Custom/community LUTs = usually free but check the source
- Sound effects from Freesound = check each file's CC license
  (CC0 = completely free; CC BY = credit required; CC BY-NC = personal only)
